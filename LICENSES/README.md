# Third-party font licenses

This folder collects the license / notice files for the bundled fonts. The extension code itself is covered by the repository's top-level LICENSE (if present).

| File | Covers |
|---|---|
| `Atkinson-Hyperlegible-OFL.txt` | `AtkinsonHyperlegible-Regular.ttf`, `-Bold.ttf`, `-Italic.ttf`, `-BoldItalic.ttf` — SIL Open Font License 1.1, © Braille Institute of America, Inc. |
| `OpenDyslexic-OFL.txt` | The OpenDyslexic typeface that `Fast_OpenDyslexic.ttf` is derived from — SIL Open Font License 1.1, © Abelardo Gonzalez. |
| `Fast-Font-NOTICE.txt` | `Fast_Sans.ttf`, `Fast_Sans_Dotted.ttf`, `Fast_Serif.ttf`, `Fast_Mono.ttf`, `Fast_OpenDyslexic.ttf` — Fast-Font has no formal license file upstream; this notice quotes the project's stated terms and links to the source. |
